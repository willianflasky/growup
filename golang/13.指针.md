#### 1.指针

- 1. 指针和地址的区别
	- 地址：就是内存地址，用字节来表示。
	- 指针：带类型的地址。

	- &：表示取地址。
	- *：表示取值。


```
package main

import "fmt"


func main()  {
	a := 10
	p1 := &a
	fmt.Printf("%T\n", p1)	// 查看指针的类型:	*int
	fmt.Printf("%v\n", p1)	// 查看针指：0xc42007e008
	fmt.Printf("%v\n", *p1) // 查看指针的值：10
}
```
#### 2.指针的使用

```
package main

import "fmt"

func main()  {
	a := [4]int{1,2,3,4}
	f1(a)	  // 不使用指针
	//f2(&a)  // 使用指针
	fmt.Println(a)
}

func f1(a [4]int){
	a[0]=100
}

func f2(a *[4]int){
	(*a)[0]=100
}

```
